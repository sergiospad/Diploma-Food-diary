Разработанное приложение включает в себя управление
пользователями, личным кабинетом, рецептами, комментариями, дневником
питания и контролем веса. Система обеспечивает регистрацию и
авторизацию пользователей, поддержку ролей пользователя, администратора
и анонимного посетителя, а также возможность редактирования персональных
данных. Авторизованный пользователь может создавать, редактировать и
удалять собственные рецепты, добавлять их в избранное, оставлять
комментарии и управлять своими записями. Администратор получает
расширенные права на модерацию рецептов и комментариев.

Клиентская часть системы реализована на языке программирования
TypeScript с использованием фреймворка Angular. Серверная часть
разработана на языке Java с применением фреймворка Spring Boot. В качестве
системы управления базой данных используется PostgreSQL
